## How to Add Wiki Pages

1. Create a new `.mdx` file in `apps/nyc/src/app/wiki/_pages/`
2. Add frontmatter at the top (title, description, category, lastUpdated)
3. Write your content in Markdown below the frontmatter
4. The page will automatically appear at `/wiki/your-file-name`

## Frontmatter Template

```
---
title: Your Page Title
description: A short summary of this page.
category: Category Name
lastUpdated: 2026-04-07
---
```

Pages are grouped by category on the wiki index and sorted alphabetically within each group.
